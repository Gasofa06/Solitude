FAQs

Frequently Asked
Questions.

  1. Homomorphic encryption is a cryptographic technique that allows computations to be performed on encrypted data. This ensures that raw data stays fully encrypted during processing, providing privacy while allowing third parties to operate with the message without knowing its content.

  2. PIR stands for Private Information Retrival and it's a protocol that allows a user to retrieve an item from a server with a database without disclosing which specific item is being retrieved.

  3. Solitude Search incorporates Blyss's code to facilitate private queries. It leverages Homomorphic Encryption to establish a PIR protocol, ensuring confidentiality in search operations.

  4. With Solitude Search, not only is there no data collection, but there's also a guarantee that all data will remain encrypted at all times. Only the user himself will have knowledge of the searches made.
